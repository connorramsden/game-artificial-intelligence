# Game Artificial Intelligence
Projects completed in GPR-340 (Game AI) at Champlain College in the Fall of 2019 under professor Dean Lawson

**Topics Covered:**
- **Steering Algorithms**
  - Arrive & Face
  - Flocking / Boids
  - Pathfinding
- **Graph Traversal Algorithms**
	- Dijkstra's
	- A*
  - Dynamic & Hierarchical Pathfinding
- **State Machines**
- **Unity + TensorFlow**
